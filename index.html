What this portal does

Buyer-facing lot cards, owner-editable data, PDF spec sheets.

DRIVN is a lightweight static app for presenting Driva green bean lots, processes, tasting notes, prices, and technical details. It is designed to feel like a clean learning portal: soft sand panels, navy command-room contrast, rounded cards, tactile borders, and compact lot metadata.
